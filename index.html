Concept Definitions
Key terms in Context-Free Grammars
📐CFG
A formal system of variables, terminals, and production rules used to generate strings in a language.
G = (V, Σ, R, S)
🔠Variables (Non-terminals)
Symbols that can be expanded using production rules. They appear on the left-hand side of productions.
S, A, B, C
🔡Terminals
Symbols that appear in the final derived string and cannot be expanded further.
a, b, 0, 1, (, )
⚙️Production Rules
Rules defining how a variable can be replaced by a sequence of variables and/or terminals.
A -> aA | a S -> aSb | ε
⬅️Leftmost Derivation
A derivation where the leftmost non-terminal in the sentential form is replaced at every step.
S ⇒ AB ⇒ aAB ⇒ aaB ⇒ aabb
➡️Rightmost Derivation
A derivation where the rightmost non-terminal in the sentential form is replaced at every step.
S ⇒ AB ⇒ AbB ⇒ Abb ⇒ aabb
🌳Parse Tree
A tree showing how a string is derived from the start variable. Internal nodes are variables; leaves are terminals or ε.
Root = start variable Leaves = terminals
∅Epsilon (ε)
The empty string. A production A → ε means variable A can derive nothing — it disappears from the sentential form.
S -> aSb | ε
🔁Sentential Form
Any intermediate string of variables and terminals produced during a derivation, starting from the start symbol.
S → aAB → aaB → aabB
